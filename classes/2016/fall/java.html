Syllabus

  1. Classes 1, 2, and 3: Basics
    • Hello World (output)
    • compilation
    • assignment
    • variable types (int, double, boolean), cast
    • integer division, increment, decrement, arithmetic operations
    • if statements
    • while loops
    • for loops
    • boolean operators
    • Strings
    • input
    • arrays and ArrayLists
    • Math.random()
    • Other math operators (sqrt, pow, etc.)
  2. Classes 4, 5, and 6: Intro to OOP (object-oriented programming)
    • classes
    • objects
    • methods
    • fields
    • public, private, static, final variables
    • inheritance
    • activities
