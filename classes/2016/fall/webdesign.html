Website Design - Fall 2016

In today’s world, everybody should have a basic understanding of some computer skills, including the basics of how a website works. High school students especially need a personal website to record their activities so when college application season comes, admission officers will be amazed by their history.

In this class, you will learn the fundamentals of website design from a coding perspective. You will become familiar with HTML and CSS, and by the end of the class you will have completed a personal website that you can build your profile on!

This class is mainly for middle school students with no prior experience. We will help your student to successfully establish his or her very first personal website and cultivate his/her love for technology!

There will be two sessions. Each session teaches the same material. Therefore, if you can’t attend a class in Session 1 you can make it up at the same class in Session 2. Session 1 will be taught by Andrew Zhou and assisted by Jeffrey Tian and Samuel Yang. Session 2 will be taught by Brandon Fu and Jason Yang. Each session will have around 10 students.

Syllabus

  1. HTML elements and inline CSS
  2. CSS colors, fonts, images, and classes (both internal and external)
  3. Rest of CSS (padding, ID, hex code, etc.)
  4. Bootstrap containers and grid; work time
  5. Bootstrap wells, buttons, and navigation bar
  6. Final work time and presentation of website
