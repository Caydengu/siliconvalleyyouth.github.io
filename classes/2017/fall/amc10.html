AMC10

  • Description
  • Class Photos

The aim of this class is to help students qualify for AIME. This class will be teaching concepts from number theory, algebra, geometry, and counting, and focuses on applying these concepts to the right problems. By the end of this class students should be able to apply the new concepts to approach harder problems. The target audience of this class should be students who can consistently score 90+ on the AMC 10 or 20+ on the AMC 8.

Dates: Fridays 9/8, 9/22, 9/29, 10/20, 10/27, 11/17, 12/1, 12/8

Time: 7:00pm–8:30pm

Location: Miller Middle School, 6151 Rainbow Dr, San Jose, CA 95129
